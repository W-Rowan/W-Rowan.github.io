4. Technical Approach

Keep Jekyll + al-folio. The theme already handles publications with BibTeX, CV from YAML, responsive layout, dark mode, SEO, and GitHub Pages deployment. We'll heavily customise the content, styling, and structure rather than rebuild from scratch.

What changes:

  • Theme colours: Clean white/light grey, subtle accent colour, modern sans-serif (Inter or similar)
  • Home page: Complete rewrite of about.md — new hero with punchy bio + rotating highlights
  • Navigation: Research, Film, Projects, CV (replacing the current About/CV/Publications)
  • Research page: New page combining publications + research narrative + awards + service
  • Film page: New page for criticism, festivals, filmmaking, programming
  • Projects page: Card-based grid with real images/logos, links to live sites
  • CV data: Complete rewrite of _data/cv.yml with all current information
  • Publications: Update papers.bib with all 4 papers + correct metadata
  • News: Fresh announcements (CVMP, PXLD, recent awards)
  • Images: Copy in real assets — profile photo, project logos, Film Fever stills
  • Social links: All current links (GitHub, Scholar, LinkedIn, email)
  • Meta/SEO: Updated title, description, OG tags

What stays:

  • Jekyll build system + GitHub Pages deployment
  • al-folio's publication engine (jekyll-scholar + BibTeX)
  • Dark mode toggle
  • Responsive layout + Bootstrap grid
  • CV layout engine (YAML-driven)
